# Lightweight Outreach Playbook

This starter kit is for planning and review. It does not send emails.

## 1. Define the project

Start with a project config that describes the market, offer, target segments, excluded segments, scoring rules, and stop conditions.

## 2. Build a reviewable lead pool

Use public, lawful, and relevant sources. Keep notes about why each lead appears to be a fit. Remove regulated, sensitive, or unclear categories before outreach.

## 3. Score and dedupe

Run the local scoring script to identify leads that are ready for human review. Dedupe by public email, website, or business name plus region.

## 4. Review before outreach

Every lead should be reviewed by a person before any private outreach system is used. Confirm business relevance, contact quality, region, and opt-out requirements.

## 5. Preview messages

Use the dry-run script to preview intent, tone, offer, and required opt-out language. This starter kit intentionally avoids sending capability.

## 6. Report weekly

Use the mock weekly report to summarize lead pool health, review queue size, excluded records, duplicates, and next actions.
